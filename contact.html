Contact Information

Emerging Languages is organized by Alex Payne. Please feel free to email him with questions or feedback.

Our host conference since 2012 is Strange Loop. Please contact their team if you have questions about conference logistics. Strange Loop is an annual conference held in autumn in St. Louis, Missouri, USA.

If you have corrections or additions (for example: a link to slides for a presentation, an updated website for a speaker, or a link to coverage of a given year's event) please submit a pull request to the GitHub repository that contains the source for this site.
